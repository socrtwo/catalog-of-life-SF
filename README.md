# Catalogue of Life Converter

Converts the Catalogue of Life database (1.2 million species) into GEDCOM genealogy format, producing over 2 million records. Explores hybridization-driven speciation patterns.

**Language:** MS Access / VBA

## Features

- Converts Catalogue of Life species database to GEDCOM format
- Handles 1.2 million+ species records
- Outputs 2 million+ genealogy-style records
- MS Access database with VBA automation
